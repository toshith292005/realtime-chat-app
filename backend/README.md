# 💬 Real-Time Chat Application

## 🧠 Overview

A real-time chat application built using WebSockets (Socket.io) that supports multi-user communication, chat rooms, typing indicators, and message persistence using MongoDB.

---

## ⚙️ Tech Stack

* Node.js
* Express.js
* Socket.io (WebSockets)
* MongoDB (Mongoose)
* HTML, CSS (Frontend)

---

## 🚀 Features

* 🔴 Real-time messaging
* 👥 Chat rooms (multiple groups)
* ✍️ Typing indicator
* 🟢 Online users list
* 💾 Message storage (MongoDB)
* ⚡ Instant updates across clients

---

## 🏗️ Architecture

Client → Socket.io Server → MongoDB

---

## ▶️ How to Run

```bash
npm install
node server.js
```

Open `index.html` in browser.

---

## 📸 Demo

* Open multiple tabs
* Join same room
* Send messages
* See real-time updates
