# 🚀 Cloud-Native Real-Time Chat Application

A full-stack real-time chat application built with modern web technologies and deployed using cloud-native practices.

---

## 🧠 Overview

This project demonstrates a scalable real-time communication system using WebSockets, containerization, and Kubernetes orchestration. It is designed to simulate production-level architecture with multiple services and deployment strategies.

---

## ⚙️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Real-Time:** Socket.io  
- **Database:** MongoDB Atlas  
- **Containerization:** Docker  
- **Orchestration:** Kubernetes  
- **Deployment:** Vercel (frontend), Render / Kubernetes (backend)

---

## 🔥 Features

- Real-time messaging using WebSockets  
- Multiple users and chat rooms  
- Persistent message storage with MongoDB  
- Scalable backend using Kubernetes (multiple replicas)  
- Containerized application using Docker  
- Clean microservices-ready architecture  

---

## 🏗️ Architecture
Frontend (React)
↓
Backend (Node.js + Socket.io)
↓
MongoDB Atlas
↓
Docker → Kubernetes (Scaling & Orchestration)
---

## 🚀 Deployment

- Frontend deployed on Vercel  
- Backend containerized using Docker  
- Kubernetes used for scaling and managing backend services
